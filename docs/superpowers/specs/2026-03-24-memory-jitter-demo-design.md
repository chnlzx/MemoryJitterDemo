# 内存抖动示例项目设计文档

## 项目概述

**项目名称**：MemoryJitterDemo

**目标**：创建一个专门用于学习Android内存抖动的示例项目，使用Java语言

**技术栈**：
- Java
- XML布局
- ViewModel + LiveData
- LeakCanary (内存泄漏检测)

---

## 项目架构

```
app/
├── data/
│   ├── model/          # 数据模型
│   └── repository/     # 数据仓库
├── ui/
│   ├── activity/       # Activity页面
│   └── adapter/        # RecyclerView适配器
├── utils/
│   └── memory/         # 内存优化工具
└── di/                 # 依赖注入
```

---

## 内存抖动示例场景

### 场景1：频繁创建对象
- 在循环中频繁创建临时对象
- 字符串拼接导致大量临时对象

### 场景2：Bitmap未回收
- 大量Bitmap未及时回收
- Bitmap内存泄漏

### 场景3：集合频繁扩容
- ArrayList频繁扩容
- HashMap频繁扩容

---

## 主要页面设计

# 终端彩虹爱心绘制设计文档

## 1. 需求概述

用户希望在终端窗口中绘制一个大大的彩虹色爱心，具体需求如下：

- **窗口类型**：终端窗口（ASCII艺术）
- **爱心尺寸**：中等（约20行）
- **颜色**：彩虹色
- **动画**：静态显示

## 2. 设计目标

创建一个Python脚本，能够在终端中显示一个彩虹色的静态爱心，支持自定义尺寸。

## 3. 架构设计

### 3.1 整体架构

```
draw_heart.py
├── calculate_heart_points()  # 计算爱心点阵
├── map_to_rainbow_color()   # 映射位置到彩虹色
└── main()                   # 主函数，输出到终端
```

### 3.2 组件详情

#### 3.2.1 calculate_heart_points()

- **输入**：爱心尺寸（行数）
- **输出**：二维列表，表示爱心点阵（True/False）
- **实现**：使用爱心方程 `(x²+y²-1)³ - x²y³ ≤ 0` 计算每个点是否在爱心内
- **注意**：需要将终端坐标系（原点在左上角）转换为数学坐标系（原点在中心）

#### 3.2.2 map_to_rainbow_color()

- **输入**：点在爱心中的位置（x, y坐标）
- **输出**：ANSI颜色代码
- **实现**：根据x坐标映射到彩虹色谱（红→橙→黄→绿→蓝→紫）
- **注意**：彩虹色映射基于点在爱心水平方向的位置，确保颜色渐变均匀

#### 3.2.3 main()

- **输入**：可选的尺寸参数
- **输出**：到终端的彩色爱心
- **实现**：遍历点阵，为每个点设置颜色并输出

## 4. 数据流

1. 用户运行脚本（可选尺寸参数）
2. calculate_heart_points() 生成点阵
3. 对于每个点，map_to_rainbow_color() 计算颜色
4. 输出带ANSI颜色代码的字符到终端

## 5. 错误处理

1. **终端颜色支持检查**：检测终端是否支持ANSI颜色代码
2. **尺寸验证**：确保尺寸参数在合理范围内（5-50行）
3. **Python版本检查**：确保Python 3.6+

## 6. 测试策略

1. **手动测试**：运行脚本，目视检查爱心形状和颜色
2. **单元测试**：测试点阵计算函数
3. **边界测试**：测试最小和最大尺寸

## 7. 使用示例

```bash
# 基本使用（默认20行）
python3 draw_heart.py

# 自定义尺寸
python3 draw_heart.py --size 30

# 查看帮助
python3 draw_heart.py --help
```
